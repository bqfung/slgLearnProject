# 第 4 阶段测试清单：敌人波次与 Boss

## 准备

1. 打开 Unity 工程 `slgLearnProject/`。
2. 等待 Unity 编译完成。
3. 点击菜单：`SLG Learn > Build Stage 04 Boss Scene`。
4. Unity 会生成并打开场景：`Assets/_Project/Scenes/Stage04_Boss.unity`。

## 运行测试

- 点击 Play 后，小队应自动前进。
- 普通敌人应向小队移动。
- 敌人靠近小队后，应按攻击频率造成减员。
- 士兵进入攻击范围后，应自动攻击敌人。
- 经过倍率门后，小队人数变化应影响清怪效率。
- 到达终点后，应能攻击 Boss。
- Boss 靠近小队后，应造成更高减员。
- Boss 被击败后，应显示 `VICTORY`。
- 小队人数归零后，应显示 `DEFEAT`，并停止前进。

## 当前可接受问题

- 当前敌人波次是场景中预摆放的对象，还不是配置驱动生成。
- Boss 没有血条，暂时通过死亡和胜利文字验证流程。
- 敌人攻击没有动画、特效和音效。
- 失败后普通敌人仍可能处于激活状态，后续流程管理阶段会统一暂停。

## 通过标准

- 敌人可以接近小队并造成减员。
- Boss 可以被攻击并被击败。
- 胜利和失败条件都能触发。
- 运行过程中没有明显报错。
