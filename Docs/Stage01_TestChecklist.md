# 第 1 阶段测试清单：玩家移动与小队系统

## 准备

1. 打开 Unity 工程 `slgLearnProject/`。
2. 等待 Unity 编译完成。
3. 点击菜单：`SLG Learn > Build Stage 01 Movement Scene`。
4. Unity 会生成并打开场景：`Assets/_Project/Scenes/Stage01_Movement.unity`。

## 运行测试

- 点击 Play 后，小队应自动向前移动。
- 按 A/D 或左右方向键，小队应左右移动。
- 按住鼠标左键左右拖拽，小队应左右移动。
- 在移动端触摸拖拽时，小队也应左右移动。
- 按 E，小队人数增加。
- 按 Q，小队人数减少。
- 人数变化后，队形应自动重新排列。
- 摄像机应跟随小队前进，并保持俯视角度。
- 小队横向移动不应超出赛道边界。

## 当前可接受问题

- 士兵临时使用胶囊体，占位表现比较简陋。
- 还没有动画、攻击、碰撞减员和 UI。
- 摄像机角度和移动手感后续可以继续调参。

## 通过标准

- 小队移动、输入、人数变化、阵型刷新、摄像机跟随全部可用。
- 控制过程中没有明显报错。
- 场景保存后能再次打开并运行。
