# 第 2 阶段测试清单：倍率门系统

## 准备

1. 打开 Unity 工程 `slgLearnProject/`。
2. 等待 Unity 编译完成。
3. 点击菜单：`SLG Learn > Build Stage 02 Gates Scene`。
4. Unity 会生成并打开场景：`Assets/_Project/Scenes/Stage02_Gates.unity`。

## 运行测试

- 点击 Play 后，小队应自动前进。
- 左右移动小队，选择不同倍率门。
- 经过 `+5` 门时，小队人数应增加 5。
- 经过 `x2` 门时，小队人数应变为当前人数的 2 倍。
- 经过 `-3`、`-6` 门时，小队人数应减少对应数量。
- 每个门只能触发一次。
- 门触发后应隐藏，避免重复触发。
- 人数变化后，阵型应自动刷新。

## 当前可接受问题

- 门的美术表现是临时方块和文字。
- 还没有做选择提示、特效、音效和 UI 数字反馈。
- 减员时暂时从队尾移除士兵，后续战斗阶段会结合受击逻辑优化。

## 通过标准

- 加法、减法、乘法门都能正确改变人数。
- 同一个门不会被多个士兵重复触发。
- 运行过程中没有明显报错。
