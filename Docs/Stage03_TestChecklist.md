# 第 3 阶段测试清单：自动战斗系统

## 准备

1. 打开 Unity 工程 `slgLearnProject/`。
2. 等待 Unity 编译完成。
3. 点击菜单：`SLG Learn > Build Stage 03 Combat Scene`。
4. Unity 会生成并打开场景：`Assets/_Project/Scenes/Stage03_Combat.unity`。

## 运行测试

- 点击 Play 后，小队应自动前进。
- 小兵接近敌人后，应自动攻击前方最近敌人。
- 敌人受到足够伤害后，应消失。
- 经过加法或乘法门后，小队人数增加，整体输出应明显变高。
- 经过减法门后，小队人数减少，整体输出应变低。
- 多个小兵同时攻击时，游戏不应明显卡顿。

## 当前可接受问题

- 当前攻击是即时伤害，没有子弹飞行表现。
- 敌人暂时不会移动和反击。
- 没有伤害数字、受击特效、音效和血条。
- 士兵索敌逻辑仍然比较简单，后续会继续优化。

## 通过标准

- 士兵可以自动找到前方敌人。
- 敌人血量归零后会被击败。
- 倍率门改变人数后能影响战斗效率。
- 运行过程中没有明显报错。
